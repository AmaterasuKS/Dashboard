# Amaterasu Dashboard

Amaterasu is a polished, interactive dashboard for monitoring financial transactions in real time. It combines advanced filtering, clear visual analytics, and a responsive UI to help teams quickly spot trends and make data-driven decisions.

## Highlights
- Advanced filters (date ranges, amounts, volumes, status, category, type)
- Interactive charts (line, bar, doughnut) with hover crosshair
- Smart sorting and “View more” pagination
- Bilingual UI (EN/RU) with locale-aware formatting
- Clean, modern design ready for client demos

## Tech Stack
- HTML, CSS, JavaScript
- Chart.js for visualizations

## Getting Started
1. Open `index.html` with Live Server (VS Code)  
2. Or open `index.html` directly in a browser

No build step required.
